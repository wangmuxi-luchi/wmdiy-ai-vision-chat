## 第 10 页：技术架构 — 核心技术点

```
【页面标题】四个核心技术突破

【技术 1：HTTP 音频管线】
自研 SilenceDetector（PCM RMS 静音检测）累积音频
800ms 静音判定用户说完 → HTTP ASR 语音转文字
无需依赖实时 WebSocket，稳定可靠

【技术 2：多模态融合推理】
ASR 文字 + 摄像头画面描述 → 融合 prompt → step-3.7-flash 一次推理
step-3.7-flash 原生支持图片+文字多模态输入
Token 预算管理防止上下文溢出

【技术 3：按需截图 + 动态帧率】
语音触发时才截图，不持续占用主线程
画面静止时降低分析频率，节约 API 成本 50%+

【技术 4：全链路 WebSocket + HTTPS 部署】
前后端单条 WebSocket 双向通信
JSON 文本 + Binary 音频同一条通道传输
腾讯云 + nginx 反向代理 + 自签名 SSL 证书
```
