### 3.2 websocket_handler.py — WebSocket 网关

| 属性 | 值 |
|------|-----|
| 路径 | `backend/websocket_handler.py` |
| 行数 | 148 |
| 状态 | ✅ 已验证连接与收发 |

**ConnectionManager** — 会话管理
- `connect(session_id, ws)` / `disconnect(session_id)` — 连接生命周期
- `send_json(session_id, data)` / `send_bytes(session_id, data)` — 发送消息
- 使用 `Dict[str, WebSocket]` 维护会话映射

**消息协议（JSON）**：

| type | data | 说明 |
|------|------|------|
| `frame` | Base64 JPEG | 视频帧 → vision_processor |
| `audio` | Base64 音频 | 音频 → audio_processor |
| `control` | `{"command": ...}` | 控制命令 |
| `ping` | - | 心跳 |
| `test_chat` | string | 文字聊天（Web 客户端用） |

**回复消息类型**：
- `connected` — 连接确认
- `user_message` / `assistant_message` — 对话文字
- `frame_analyzed` — 画面分析结果
- `control_ack` / `pong` — 命令确认/心跳回复
- 二进制数据 — TTS MP3 音频

## 5. 数据流（一次典型对话回合）

```
Step  组件             动作                        数据格式
───  ────────────     ──────────────────────       ─────────────────
 1   Flutter Camera   打开摄像头，开始预览           MediaStream
 2   Flutter Camera   Canvas 定时截帧（默认 1s）    JPEG Base64
 3   Flutter Audio    录音 3s 片段                  PCM 16kHz 16bit
 4   Flutter WS       → 发送帧                      {"type":"frame","data":"<base64>"}
 5   Flutter WS       → 发送音频                    {"type":"audio","data":"<base64>"}
 6   FastAPI WS       接收 → 路由到对应处理器
 7   Cost Control     帧率控制（静态降频） / 静音检测
 8   Vision API       帧 → GPT-4 Vision → 画面描述   文本（中文，1-2句）
 9   Whisper API      音频 → Whisper-1 → 文本       文本
10   Agent            deepagents 融合(画面+语音) → 推理  → 回复文本
11   TTS API          回复文本 → TTS-1 → MP3        二进制音频
12   FastAPI WS       ← 发送文字回复                  {"type":"assistant_message","text":"..."}
13   FastAPI WS       ← 发送 TTS 音频                二进制数据
14   Flutter          显示文字气泡 + 播放语音
```
